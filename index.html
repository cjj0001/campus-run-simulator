📚 核心算法说明

1. 路径生成算法

  • • 贝塞尔曲线平滑：使用三阶贝塞尔曲线对路径进行平滑处理，避免锐角
  • • 等距采样：按固定时间间隔对路径进行采样，确保点间距均匀
  • • 点位绕行：自动计算经过打卡点的最优绕行路线

2. 速度模拟算法

  • • S型速度曲线：起步→加速→匀速→减速的自然速度变化
  • • 随机波动：基于正态分布的速度波动，模拟真实跑步节奏
  • • 弯道减速：在转弯处自动降低速度，增加真实感

3. GPS漂移模拟

  • • 高斯噪声：添加符合正态分布的位置噪声
  • • 连续漂移：使用Perlin噪声生成连续的漂移轨迹
  • • 精度衰减：模拟信号弱区域的定位精度下降

4. 数据自洽性校验

  • • 时间戳同步：确保距离、速度、时间三者数学自洽
  • • 加速度限制：限制最大加速度，避免不真实的突变
  • • 异常检测：自动检测并修正不合理的数据点

📖 使用文档

快速开始

  1. 在地图上点击添加起点、途经点和终点，或点击"加载预设点位"
  2. 调整各项参数（距离、速度、GPS漂移等）
  3. 点击"生成路径预览"查看规划的路线
  4. 点击"开始模拟"启动实时路径模拟
  5. 模拟完成后可导出各种格式的轨迹数据
